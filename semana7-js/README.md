# 📊 Librería de Funciones - Semana 7

Este proyecto implementa una librería de funciones en JavaScript aplicada a una plataforma de **crowdfunding**.
Permite gestionar campañas, formatear datos, validar información y calcular resultados utilizando funciones reutilizables.

---

## 🎯 Objetivo

Aplicar funciones para:

* Organizar lógica del programa
* Reutilizar código
* Separar responsabilidades
* Generar reportes estructurados

---

## 📌 ¿Qué hace el programa?

* Muestra un listado de campañas
* Formatea la información de cada campaña
* Filtra campañas activas
* Calcula el total recaudado
* Genera un reporte final

---

## 🧠 Conceptos usados

### 🔹 Funciones

* Arrow functions (`=>`)
* Funciones puras
* Funciones de validación

### 🔹 Parámetros

* Parámetros por defecto

### 🔹 Otros

* Template literals
* Bucles (`for...of`)
* Operador nullish (`??`)
