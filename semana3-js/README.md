# 💰 Calculadora de Campañas - Semana 3

Este proyecto es una calculadora sencilla hecha en JavaScript basada en una plataforma de crowdfunding (tipo Kickstarter).

La idea es simular cómo se calcula el dinero de una campaña, cuánto falta para la meta y algunas condiciones como si la campaña es popular o si aplica algún beneficio.

---

## 📌 ¿Qué hace el programa?

- Calcula el dinero total recaudado
- Muestra cuánto falta para llegar a la meta
- Saca el promedio de aportes
- Aplica operaciones como suma, resta, multiplicación y división
- Usa condiciones para verificar estados de la campaña

---

## 🧠 Qué aprendí

En este proyecto usé:

- Operadores matemáticos (`+`, `-`, `*`, `/`, `%`)
- Asignación compuesta (`+=`, `*=`)
- Comparaciones (`===`, `!==`, `>=`)
- Operadores lógicos (`&&`, `||`, `!`)
- Uso de `console.log()` para mostrar resultados
