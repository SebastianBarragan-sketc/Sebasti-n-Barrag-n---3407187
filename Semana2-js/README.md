# Proyecto Semana 2 – Ficha de Datos del Dominio

## Dominio
Crowdfunding

## Descripción
Este proyecto consiste en crear un script en JavaScript que se ejecuta con Node.js y muestra en la consola una ficha de datos sobre una entidad del dominio asignado.

En este caso el dominio es **Crowdfunding** y la entidad utilizada es una **campaña de financiamiento para un videojuego indie**.

El programa muestra información básica de la campaña como su nombre, categoría, meta de dinero y estado, además de verificar los tipos de datos de algunas variables y realizar una conversión de tipos.

---

## Objetivo del proyecto

Practicar los conceptos aprendidos en la semana 2:

- Uso de variables con `const`
- Tipos de datos en JavaScript
- Uso de `typeof`
- Conversiones de datos
- Uso de `console.log()`
- Uso de template literals

## Conceptos aplicados

En este proyecto se utilizaron los siguientes conceptos de JavaScript:

- Declaración de variables con `const`
- Uso de tipos de datos `string`, `number`, `boolean` y `null`
- Verificación de tipos con `typeof`
- Conversión de número a texto usando `String()`
- Uso de `console.log()` para mostrar información en consola
- Uso de template literals para insertar variables en texto
